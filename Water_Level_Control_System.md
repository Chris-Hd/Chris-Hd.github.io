# Water Level Control System  
**Microprocessors & Embedded Systems Project**

This project provided hands-on experience with microprocessors, focusing on system design, programming, and hardware integration. The system was built around learning microprocessor operation and control logic, including working with the 8085 microprocessor and ARM-based controllers.
